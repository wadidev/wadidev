## Proyectos destacados

### ECG Explorer

Aplicación educativa desarrollada con Python y Streamlit para visualizar señales ECG, aplicar filtrado básico, detectar candidatos a picos R y calcular métricas como intervalos RR y frecuencia cardíaca estimada.

Tecnologías: Python, Streamlit, NumPy, Pandas, SciPy, WFDB, Matplotlib y Pytest.

### Dashboard de constantes vitales

Dashboard educativo con datos simulados de constantes vitales. Permite visualizar frecuencia cardíaca, presión arterial, temperatura y saturación de oxígeno, con clasificaciones básicas orientativas.

Tecnologías: Python, Streamlit, Pandas, NumPy y Matplotlib.

### Portfolio personal

Portfolio web donde presento mi perfil, formación, proyectos y áreas de interés dentro de la ingeniería biomédica y la programación.

Tecnologías: HTML, CSS y JavaScript.

### WadiDev
