Dudas frecuentes

Antes de empezar

Respuestas rápidas para que sepas exactamente qué incluye el servicio.

¿El dominio y el hosting están incluidos?

La publicación básica puede hacerse en GitHub Pages. Si quieres dominio propio, te ayudo a configurarlo y te indico el coste externo antes de contratarlo.

¿La web funciona en móvil?

Sí. Todas las webs se diseñan responsive para móvil, tablet y ordenador, porque la mayoría de clientes entran desde el teléfono.

¿Puedo pedir cambios?

Sí. El proyecto incluye revisión del contenido y ajustes básicos antes de publicar para que la web encaje con tu negocio.

¿Aparecerá en Google?

La web se entrega con SEO básico/local: títulos, descripción, estructura clara y datos preparados para que Google la entienda mejor.
